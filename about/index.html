At first, I did not know how to properly name CSS classes and structure HTML tags so that my code is meaningful and will not cause unnecessary problems in the future. I found BEM when I was searching on Google, and I think it is a good enough method because it provides a clear structure on how to manage HTML tags and CSS classes. The docs gives examples that help me understand how to apply the method. In addition, I can inspect the website's HTML and CSS using Firefox's DOM and Style Inspector to understand how BEM team implements the methodology. How they name classes and organize the elements. Also, I think the naming scheme is cool because it is detailed.
